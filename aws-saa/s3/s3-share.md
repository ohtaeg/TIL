# S3 공유
- S3 공유는 3가지 방법으로 가능
1. Bucket Policy / IAM
   - 프로그램 엑세스만 가능 (콘솔에서 불가능)
   - `버킷 단위`에서 적용
2. ACL
   - 프로그램 엑세스만 가능 (콘솔에서 불가능)
   - `파일 단위`
3. IAM Cross Account
   - 콘솔 / 프로그램 엑세스 가능
   - 하나의 계정에서 다른 계정에게 권한을 할당

<br>

## S3 Cross Region Replication
- 다른 Region으로 복제 가능
- 버저닝이 활성화 되어 있어야함 (원본, 대상 모두)
- 동일한 Region으로 복제 불가능
- 복제 기능 활성화 전의 데이터는 복제되지 않음
- 버전 삭제 혹은 파일 삭제는 복제되지 않음
